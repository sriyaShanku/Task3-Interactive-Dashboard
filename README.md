# 📊 Task 3 – Deep-Dive Analysis &amp; Interactive Dashboard

## 📌 Overview
This project focuses on creating an interactive business dashboard using Looker Studio to visualize sales and profit insights from the Superstore dataset.

## 🎯 Objectives
- Build an interactive dashboard
- Visualize KPIs and trends
- Enable business decision-making through data visualization

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

### 🔹 Visualizations
- Sales by Region
- Sales by Category
- Profit by Category
- Sales by Segment

### 🔹 Interactive Filters
- Region Filter
- Category Filter
- Segment Filter

## 🛠️ Tools Used
- Looker Studio (Google Data Studio)
- Python
- Pandas
- CSV Dataset

## 📈 Key Insights
- West region generated the highest sales
- Technology category performed best
- Discounts negatively impacted profits
- Sales trends varied across categories and regions
